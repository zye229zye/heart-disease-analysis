#Graphical Analysis

A scatter plot illustrates the relationship between serum cholesterol levels and age, differentiated by heart disease presence.

library(ggplot2)

# Create a scatter plot of age vs cholesterol, colored by target
ggplot(data, aes(x = age, y = chol, color = as.factor(target))) +
  geom_point(alpha = 0.7) +
  labs(
    title = "Scatter Plot of Age vs Cholesterol Level",
    x = "Age",
    y = "Cholesterol Level (mg/dl)",
    color = "Heart Disease Presence"

  theme_bw()

#Regression Analysis

This section employs a generalized linear model to examine how age, serum cholesterol levels, and chest pain type affect the likelihood of heart disease. # Fit the primary regression model

mod <- glm(
  target ~ age + chol + cp,
  data = data
)

# Display the regression table
tbl_regression(mod) |>
  add_global_p()

#Secondary Model

A binary outcome model is fitted to analyze the relationship between the same predictors and the presence of heart disease, displaying odds ratios for interpretation. # Fit the secondary regression model

binary_mod <- glm(
  I(target == 1) ~ age + chol + cp,
  data = data,
  family = binomial()
)

# Display the regression table with exponentiated coefficients
tbl_regression(binary_mod, exponentiate = TRUE) |>
  add_global_p()
